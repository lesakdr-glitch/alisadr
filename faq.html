🚀 Общие вопросы

Что такое Luno Client?

Luno Client - это модификация для Minecraft, которая добавляет множество полезных функций для улучшения игрового процесса. Клиент включает в себя боевые модули, визуальные улучшения, модули движения и многое другое.

Бесплатен ли Luno Client?

Да, Luno Client полностью бесплатен для скачивания и использования. Мы не берем плату за основные функции клиента.

Какие версии Minecraft поддерживаются?

В настоящее время Luno Client поддерживает Minecraft 1.16.5. Мы планируем добавить поддержку других версий в будущих обновлениях.

💾 Установка и настройка

Как установить Luno Client?

Установка очень простая:

  • Убедитесь, что у вас установлена Java 17+
  • Скачайте лоадер с нашего сайта
  • Запустите лоадер и следуйте инструкциям
  • Клиент будет установлен в папку versions вашего Minecraft
  • Запустите игру через созданный профиль

Нужна ли Java для работы клиента?

Да, для работы Luno Client требуется Java 17 или выше. Это важное требование для корректной работы всех функций клиента.

Как сбросить настройки клиента?

Удалите папку "luno" в директории .minecraft. При следующем запуске клиент создаст новые настройки по умолчанию.

Где устанавливается клиент?

Лоадер автоматически устанавливает клиент в папку .minecraft/versions/ и создает соответствующий профиль в лаунчере. Вам не нужно ничего делать вручную.

🎮 Использование

Как открыть ClickGUI?

По умолчанию ClickGUI открывается клавишей RShift. Вы можете изменить эту клавишу в настройках клиента.

Как настроить кейбинды?

В ClickGUI найдите нужный модуль, кликните по нему правой кнопкой мыши и нажмите нужную клавишу для привязки.

Как сохранить конфигурацию?

Все настройки автоматически сохраняются при выходе из игры. Вы также можете создать отдельные конфиги через команды в чате.

🛡️ Безопасность

Безопасен ли Luno Client?

Да, Luno Client полностью безопасен. Мы не собираем личные данные и не содержим вредоносного кода. Однако всегда скачивайте клиент только с официального сайта.

Могу ли я получить бан за использование клиента?

Использование любых модификаций может привести к блокировке на серверах, которые их запрещают. Всегда проверяйте правила сервера перед использованием клиента.

Обнаруживают ли античиты Luno Client?

Мы постоянно работаем над обходами популярных античитов, но 100% гарантии обхода дать не можем. Используйте клиент осторожно.

🔧 Технические проблемы

Клиент не запускается, что делать?

Попробуйте следующие решения:

  • Убедитесь, что у вас установлена Java 17+ (не Java 8!)
  • Проверьте, что лоадер не поврежден
  • Запустите лоадер от имени администратора
  • Отключите антивирус временно
  • Проверьте, что Minecraft лаунчер закрыт

Игра крашится при использовании клиента

Это может быть связано с:

  • Конфликтом с другими модами
  • Недостатком оперативной памяти
  • Устаревшими драйверами видеокарты
  • Несовместимостью с версией Minecraft

Низкий FPS с клиентом

Для улучшения производительности:

  • Отключите ненужные визуальные модули
  • Уменьшите дальность рендера
  • Используйте OptiFine
  • Выделите больше RAM для Minecraft
