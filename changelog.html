История обновлений Luno Client

Версия 2.1 - 12 декабря 2025

🎨 Новые функции:

  • Добавлен современный HUD с градиентными эффектами
  • Новая система аватаров с локальным хранением
  • Улучшенный ClickGUI с анимациями
  • Система тем (светлая/темная) для веб-сайта

⚡ Улучшения:

  • Оптимизирована производительность KillAura
  • Улучшена стабильность Fly модуля
  • Обновлен дизайн всех HUD элементов
  • Улучшена совместимость с серверами

🐛 Исправления:

  • Исправлен краш при использовании Scaffold
  • Исправлена проблема с отображением кейбиндов
  • Исправлены ошибки в Target HUD
  • Исправлена работа с эффектами зелий

Версия 2.0 - 1 декабря 2025

🚀 Крупные изменения:

  • Полностью переписанный движок клиента
  • Новая система модулей с улучшенной архитектурой
  • Обновленный интерфейс с современным дизайном
  • Улучшенная система обходов античитов

🎯 Новые модули:

  • AutoTotem - автоматическое использование тотемов
  • CrystalAura - автоматическая установка кристаллов
  • HoleFiller - заполнение дыр блоками
  • AntiBot - защита от ботов

🔧 Технические улучшения:

  • Поддержка Minecraft 1.16.5 (MCP)
  • Требуется Java 17+ для оптимальной работы
  • Новый лоадер для автоматической установки
  • Установка в папку versions Minecraft
  • Улучшенная система конфигураций
  • Новый API для разработчиков
  • Оптимизация использования памяти
