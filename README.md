# Luno Client Website

Современный сайт для Luno Client с поддержкой темной и светлой темы.

## Функции

### ✨ Основные возможности:
- **Темная/Светлая тема** - переключение с сохранением в localStorage
- **Рабочие кнопки загрузки** - автоматическая загрузка JAR файла
- **Демо видео** - открытие в новой вкладке
- **Уведомления** - красивые всплывающие уведомления
- **Адаптивный дизайн** - работает на всех устройствах
- **Анимации** - плавные переходы и эффекты
- **Полная навигация** - все страницы функциональны и связаны
- **Правовые страницы** - политика конфиденциальности, условия использования
- **Поддержка пользователей** - FAQ, контакты, changelog

### 🎨 Темы:
- **Темная тема** (по умолчанию) - для геймеров
- **Светлая тема** - для дневного использования
- Автоматическое переключение скриншотов в зависимости от темы

### 📱 Адаптивность:
- Мобильное меню
- Оптимизированные размеры для планшетов
- Ретина дисплеи

## Настройка

### 1. Ссылки для загрузки:
В `script.js` измени ссылки:
```javascript
this.downloadUrl = './LunoLoader.exe'; // Лоадер для установки клиента
this.demoUrl = 'https://www.youtube.com/watch?v=your-demo-video';
```

### 2. Добавь изображения:
Создай папку `images/` и добавь скриншоты:
- `screenshot1.png` / `screenshot1-light.png` - ClickGUI
- `screenshot2.png` / `screenshot2-light.png` - Combat модули  
- `screenshot3.png` / `screenshot3-light.png` - HUD элементы

### 3. Настрой статистику:
В `script.js` измени начальные значения:
```javascript
const downloads = localStorage.getItem('downloads') || '127';
const users = '1250';
const bypass = '95';
```

## Использование

### Переключение темы:
- Кнопка в правом верхнем углу навигации
- Автоматическое сохранение выбора
- Разные скриншоты для каждой темы

### Загрузка файла:
- Кнопки "Скачать Loader" работают автоматически
- Показывают уведомление о начале загрузки
- Скачивают лоадер, который устанавливает клиент в versions/
- Требуется Java 17+ для работы клиента
- Обновляют статистику загрузок

### Демо видео:
- Кнопка "Смотреть демо" открывает видео в новой вкладке
- Показывает уведомление

## Кастомизация

### Цвета:
Измени CSS переменные в `:root`:
```css
--primary: #6366f1;
--secondary: #8b5cf6;
--accent: #06b6d4;
```

### Анимации:
Настрой скорость в CSS:
```css
--transition-fast: 0.15s ease;
```

### Уведомления:
Измени время показа в `script.js`:
```javascript
setTimeout(() => this.hideNotification(notification), 3000); // 3 секунды
```
